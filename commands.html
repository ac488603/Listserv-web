Listserv Command Reference

Command Format Description
ls print data structure to console
clear clear console output
create <student name> reate new student
mkorg <organization name> create new organization
del <name> delete student or organization
cd <organization name/..> changes current organization
subscribe <list> <organization/student> subscribes student or org to list
unsubscribe <list> <organization/student> unsubscribes student or org from list
notify <list> <message> notifies all users subscribed to list
